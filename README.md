## Peluncuran Produk Baru Startup

Aplikasi desktop berbasis Python untuk membantu pengambilan keputusan strategis
menggunakan metode Decision Tree Analysis (DTA).

---

## Cara Instalasi
1. Pastikan Python 3.8+ terinstal
2. Clone atau download repository ini
3. Masuk ke folder project
4. Install dependencies dengan perintah:
   pip install -r requirements.txt

---

## Dependencies
- Python 3.8+
- pandas
- matplotlib
- networkx
- tkinter

---

## Cara Menjalankan
Jalankan perintah berikut melalui terminal atau command prompt:
python main.py
